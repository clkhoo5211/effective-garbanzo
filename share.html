iLoveIMG

https://www.iloveimg.com/zh-cn
图片

免费图片处理网站，我经常用来压缩图片，压缩效果会比 Tinypng 更好。链式处理功能非常好用。

TinyPNG

https://tinypng.com/
图片

国际上比较出名的压缩图片工具，过去的书本中知晓的这个网站，但是有图片大小限制，也有数量限制。挺好的压缩图片替代品。

removebg

https://www.remove.bg/zh
图片AI

AI 抠图工具，特点是界面简洁，效果比 windows 自带的去除背景会更好一些，缺点是有免费有导出大小限制

Neumorphism

https://neumorphism.io/
CSSGithub

Neumorphism 是一种极具鲜明特色的设计风格，总能给新人一种惊叹感。但是应用并不广泛，实际开发也会有很多不好的点。更多是做一个留念。

Nucleo

https://nucleoapp.com/app/?library=core
Icons

图标库，网站整体很干净。独特的是，它有一类像素风格的图标，很有意思。它的 icon 也很多，还有 grass 风格的。

Loadership

https://www.loadership.com/
CSSGithub

13种 Loader 样式，在线调整使用。在 AI 时代可能意义小很多了。

CSS Buttons

https://cssbuttons.io/
CSS

有不少按钮样式。跟 Neumorphism 有一些关系。

Uiverse

https://uiverse.io/
CSS

不可错过的参考库，有很多交互效果，包括热门的 Button、Loader 设计，内容丰富，还在不断更新。

Magic UI

https://magicui.design/
组件Github

shadcn/ui 伴生库，有很多实用设计。取用也很方便，可以自己复制独立的代码。

Ai lman

https://aliimam.in/docs/backgrounds/shader-rgb
组件

类 Magic UI，但是感觉差点意思。可作为看完 Magic 意犹未尽时闲看。

网站集

404s.design
